# Stripe Sessions 24 Clone

You can view the live version of the project [here](https://stripe-sessions24-clone.netlify.app/).

## Description

This project is a replica of the Stripe Sessions 24 page dated December 2023. It was created for the purpose of practicing and improving web development skills. The page includes a variety of components and styles that mimic the design and functionality of the original site.

## Technologies Used

- React
- TypeScript
- SCSS
